# 教务管理系统（Swing+MySQL）

#### 功能
目前有教师和学生两种身份，根据不同的账户登录，跳转的相应的页面
教师：教学安排查询、录入成绩、导出成绩
学生：课表信息查询、选课、成绩查询

#### 所用技术
Swing
JDBC连接数据库
